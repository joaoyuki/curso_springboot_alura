Aula 02 - Spring Boot Actuator


Spring boot actuator é uma API do spring, que possui alguns endpoints que nos devolvem informações da nossa aplicação.

Precisamos adicionar no POM:

<dependency>
<groupId>org.springframework.boot</groupId>
<artifactId>spring-boot-starter-actuator</artifactId>
</dependency>
