Aula 7 - Simplificando a paginação e ordenação


O Spring tem um recurso no qual podemos passar um Pageable direto do Controller para a JPA. Mas para que isso funcione, precisamos ativar esse recurso na classe principal do sistema:

@EnableSpringDataWebSupport

@SpringBootApplication
@EnableSpringDataWebSupport // Permite passar um objeto Pageable direto do controller para o SpringDataJPA
public class ForumApplication {

public static void main(String[] args) {
SpringApplication.run(ForumApplication.class, args);
}

}


        // Nesse exemplo, passamos direto um Pageable do spring e temos que habilitar na classe ForumApplication essa funcionalidade
        @GetMapping("/exemplo02")
        public Page<TopicoDto> listaUsandoPagebleNoRequest(
                @PageableDefault(sort = "id") Pageable paginacao) {
            Page<Topico> topicos = topicoRepository.findAll(paginacao);
            return TopicoDto.converter(topicos);            
        }


Podemos também usar a anotação @PageableDefault e deixarmos parâmetros padrões, para o caso do usuário não passar nada e assim evirtamos da aplicação quebrar.
