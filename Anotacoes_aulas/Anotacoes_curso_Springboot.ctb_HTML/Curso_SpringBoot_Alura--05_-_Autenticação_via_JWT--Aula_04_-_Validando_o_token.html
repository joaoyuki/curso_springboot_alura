Aula 04 - Validando o token


Na nossa classe TokenService, temos que criar um método que vai validar se o token é valido ou não:

    boolean isTokenValido(String token) {
        
        try {
            Jwts.parser().setSigningKey(this.secret).parseClaimsJws(token);
            return true;
        }catch(Exception e){
            return false;
        }
    }


Agora que sabemos como validar um token, temos que chamar ele dentro na nossa classe Filter, mas temos um problema, como é uma classe de filtro, não conseguimos fazer um @Autowired, então precisamos injetar a classe TokenService através do construtor:

public class AutenticacaoViaTokenFilter extends OncePerRequestFilter{

    
    private TokenService tokenService;

    public AutenticacaoViaTokenFilter(TokenService tokenService) {
        this.tokenService = tokenService;
    }
    
    @Override
    protected void doFilterInternal(HttpServletRequest request, HttpServletResponse response, FilterChain fc) throws ServletException, IOException {
        
        String token = recuperarToken(request);
        boolean valido = tokenService.isTokenValido(token);
        System.out.println(valido);
        
        fc.doFilter(request, response); // O doFilter indica para o Spring que já alteramos tudo o que queríamos na requisição
        
    }
