Aula 02 - utilizando cache


Para utilizarmos o módulo de cache do spring, temos que adicionar no POM

<dependency>
<groupId>org.springframework.boot</groupId>
<artifactId>spring-boot-starter-cache</artifactId>
</dependency>

É necessário configurarmos um provedor de cache, mas, caso não especifiquemos um, o Spring vai salvar tudo em memória utilizando o provedor padrão em memória utilizando um HashMap

Também precisamos habilitar o uso de cache na classe principal da nossa aplicação:

@EnableCaching // Habilita o uso de cache na aplicação

@SpringBootApplication
@EnableSpringDataWebSupport // Permite passar um objeto Pageable direto do controller para o SpringDataJPA
@EnableCaching // Habilita o uso de cache na aplicação
public class ForumApplication {

public static void main(String[] args) {
SpringApplication.run(ForumApplication.class, args);
}

}


Agora só precisamos adicionar a anotação @Cacheble no método que queremos salvar o retorno em cache

@GetMapping
        @Cacheable(value = "ListaDeTopicos") // No value colocamos um indicador único para esse cache
public Page<TopicoDto> lista(@RequestParam(required = false) String nomeCurso, 
                @RequestParam int pagina, 
                @RequestParam int quantidade,
                @RequestParam String nomeDoCampoParaOrdenacao) {
            
            // Pageble é uma interface do spring
            Pageable paginacao = PageRequest.of(pagina, quantidade, Sort.Direction.ASC, nomeDoCampoParaOrdenacao);
            
if (nomeCurso == null) {
Page<Topico> topicos = topicoRepository.findAll(paginacao);
return TopicoDto.converter(topicos);
} else {
Page<Topico> topicos = topicoRepository.findByCursoNome(nomeCurso, paginacao);
return TopicoDto.converter(topicos);
}
}


É importante definir um ID para aquele cache, pois podemos ter mais de um cache na aplicação, para isso, basta adicionar o parâmetro value = “NomeDoId”

O Spring também é inteligente o suficiente para identificar a mudança no parâmetros e assim saber se estamos fazendo uma pesquisa já feita ou se é uma nova pequisa com outros parâmetros
