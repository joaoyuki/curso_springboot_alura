Aula 05 - Conhecendo o Spring Boot Admin


O Springboot admin não é uma ferramenta criada pela equipe do spring, mas sim por uma empresa Alemã, chamada codecentric

https://codecentric.github.io/spring-boot-admin/current/

Seguindo o guia, precisamos criar um novo projeto spring boot, e adicionar a dependência do spring boot admin nele. Depois só precisamos subir esse projeto

No nosso projeto principal, precisamos indicar que tem outro projeto fazendo um monitoramento, para isso, precisamos adicionar no POM a dependência do Spring boot admin:

<dependency>
<groupId>de.codecentric</groupId>
<artifactId>spring-boot-admin-starter-client</artifactId>
<version>2.1.4</version>
</dependency>


E também precisamos indicar qual é a URL no application.properties:

# Spring boot admin server
spring.boot.admin.client.url=http://localhost:8081


Basta subir os dois projetos e acessar: http://localhost:8081/
