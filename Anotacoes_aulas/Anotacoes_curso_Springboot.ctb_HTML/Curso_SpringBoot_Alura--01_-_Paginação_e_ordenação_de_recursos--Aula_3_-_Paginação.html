Aula 3 - Paginação


Para adicionarmos o recurso de paginação na nossa API, precisamos receber alguns parâmetros, como a página e o numero de registros que gostariamos de pesquisar.

        //@RequestParam Indicamos par ao spring que esse parâmetro vem da URL e é obrigatório
@GetMapping
public List<TopicoDto> lista(@RequestParam(required = false) String nomeCurso, 
                @RequestParam int pagina, 
                @RequestParam int quantidade) {
if (nomeCurso == null) {
List<Topico> topicos = topicoRepository.findAll();
return TopicoDto.converter(topicos);
} else {
List<Topico> topicos = topicoRepository.findByCursoNome(nomeCurso);
return TopicoDto.converter(topicos);
}
}


A anotação @RequestParam indica para o controller que estamos esperando aquele parâmetro na URL

Para criarmos uma paginação, precisamos usar a interface Pageable do spring:
Pageable paginacao = PageRequest.of(pagina, quantidade);

Como estamos utilizando SpringData para fazer a conexão com o banco de dados, os métodos do springData já suportam um parâmetro do tipo Pageable

@GetMapping
public Page<TopicoDto> lista(@RequestParam(required = false) String nomeCurso, 
                @RequestParam int pagina, 
                @RequestParam int quantidade) {
            
            // Pageble é uma interface do spring
            Pageable paginacao = PageRequest.of(pagina, quantidade);
            
if (nomeCurso == null) {
Page<Topico> topicos = topicoRepository.findAll(paginacao);
return TopicoDto.converter(topicos);
} else {
Page<Topico> topicos = topicoRepository.findByCursoNome(nomeCurso, paginacao);
return TopicoDto.converter(topicos);
}
}


public interface TopicoRepository extends JpaRepository<Topico, Long> {

Page<Topico> findByCursoNome(String nomeCurso, Pageable paginacao);

}
